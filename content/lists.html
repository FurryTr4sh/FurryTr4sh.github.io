Списки

Нумерованные списки

Нумерованный список начинается с тега <ol>, а каждый пункт указывается с помощью тега <li>.

Вот пример нумерованного списка:

<html>
<head>
<title>Первая страница</title>
</head>
<body>
<ol>
<li>Первый пункт</li>
<li>Второй пункт</li>
<li>Третий пункт</li>
</ol>
</body>
</html>

Результат:



Пункты списка будут автоматически пронумерованы начиная с единицы.
Однако Вы всегда можете начать нумерацию списка с любого числа, указав в теге <li> атрибут value с любым числовым значением.

Вы можете использовать нумерацию:
  1. Арабскими цифрами (по умолчанию). При необходимости – указав в теге <li> атрибут type со значением 1;

  2. Заглавными или строчными латинскими буквами, указав в теге <li> атрибут type со значением A или a соответственно;

  3. Заглавными или строчными римскими буквами, указав в теге <li> атрибут type со значением I или i соответственно.

Ненумерованные списки

Ненумерованный список начинается с тега <ul>.


<html>
<head>
<title>Первая страница</title>
</head>
<body>
<ul>
<li>Первый пункт</li>
<li>Второй пункт</li>
<li>Третий пункт</li>
</ul>
</body>
</html>

Результат:



Каждый пункт списка будет отмечен маркером.
Вы можете изменить маркер, указав в теге <li> атрибут type с любым из следующих типов маркеров:
  • disc (по умолчанию). При необходимости – указав в теге <li> атрибут type со значением disc;

  • circle;

  • square.
