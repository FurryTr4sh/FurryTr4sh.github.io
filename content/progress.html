Полоса выполнения

Элемент <progress> используется для создания полосы выполнения на веб-странице.

Атрибуты элемента <progress>:

Value – указывает степень заполненности полосы прогресса.

Max – задаёт максимальное значение, при достижении которого полоса прогресса будет заполнена.


Пример:

Статус: <progress value="35" max="100"></progress>

Результат:


Динамическое заполнение полоски прогресса можно реализовать при помощи языка JavaScript.
