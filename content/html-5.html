Прогресс завершения курса:

HTML5

Начав писать HTML5-документ, из всех нововведений первым делом Вы заметите объявление типа документа:


<!DOCTYPE html>

Был упрощён элемент объявления кодировки символов (charset):


<meta charset="utf-8">

Новые элементы в HTML5:

<article>, <aside>, <audio>, <canvas>, <datalist>, <details>, <embed>, <header>, <footer>, <nav>, <output>, <progress>, <section>, <video> и это ещё не всё!


Новое в HTML5

Формы:

  • Спецификация Web Forms 2.0 расширила функциональные возможности форм, стало возможным создание более продвинутых страниц;
  • добавлены элементы выбора даты и цвета;
  • новые поля для ввода: электронная почта, поиск и URL-адрес;
  • добавлена поддержка формами методов PUT и DELETE.

Встроенные API (от англ. Application Programming Interfaces – Интерфейсы программирования приложений):

  • функция «drag & drop»;
  • проигрывание аудио- и видеофайлов;
  • автономные веб-приложения;
  • история;
  • локальное сохранение;
  • геолокация;
  • веб-сообщения.
