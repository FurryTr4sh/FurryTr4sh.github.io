Типы элементов

В HTML большинство элементов – это так называемые блочные или строчные элементы.

Блочные элементы начинаются с новой строки.

Например: <h1>, <p>, <div>, <li> и т. д.

Строчные элементы обычно пишутся без переноса строки и могут находиться внутри других элементов.

Например: <b>, <a>, <img>, <span> и т. д.

Элемент <div> – блочный элемент, часто используемый как контейнер для других элементов HTML.

При использовании с CSS, элемент <div> может задавать стиль блокам контента:


<html>
<body>
<h1>Заголовок</h1>
<div style="background-color: green; color: white; padding: 20px;">
<p>Это параграф вашего контента.<p>
<p>Это ещё один элемент контента.<p>
</div>
</body>
</html>

Результат:


Элемент <span> – строчный элемент, часто используемый как контейнер для текста.

При использовании с CSS, элемент <span> может задавать стиль отдельных частей текста:


<html>
<body>
<h2>Некоторое
<span style="color: red;">важное</span>
сообщение
</h2>
</body>
</html>

Результат:


Итак:
Элемент <div> позволяет определить блочную секцию документа.
Элемент <span> позволяет определить строчную секцию документа.
