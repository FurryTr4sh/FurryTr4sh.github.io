Форматирование текста

Элементы форматирования

В HTML используется ряд элементов для определения стиля текста.

Элементы форматирования были созданы для отображения разных стилей текста:

<html>
<head>
<title>Первая страница</title>
</head>
<body>
<p>Обычный текст</p>
<p><b>Текст жирным шрифтом</b></p>
<p><big>Текст крупным шрифтом</big></p>
<p><i>Текст курсивом</i></p>
<p><small>Текст мелким шрифтом</small></p>
<p><strong>Важный текст</strong></p>
<p><sub>Подстрочный</sub> текст</p>
<p><sup>Надстрочный</sup> текст</p>
<p><ins>Подчёркнутый текст</ins></p>
<p><del>Зачёркнутый текст</del></p>
</body>
</html>

Параграфы в примере отформатированы по-разному, чтобы продемонстрировать назначение каждого тега:



Браузеры отображают <strong> как <b>.
